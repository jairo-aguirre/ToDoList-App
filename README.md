# ToDoList-App
To Do List App

## Description
This is a simple To Do List application built with Python and Tkinter. It allows users to add, delete, and mark tasks as completed. The tasks are stored in a SQLite database, and the application has a user-friendly interface.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- View all tasks
- View completed tasks
- View pending tasks
- Search tasks
- Sort tasks by date
- Sort tasks by status
- Sort tasks by priority
- Sort tasks by name
- Sort tasks by category
- Sort tasks by due date

## Requirements
- Python 3.x
- Tkinter
- SQLite3
- Pillow (for image handling)
- ttkbootstrap (for modern UI)
